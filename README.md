# 🎉 Happy Birthday App

A beautiful, animated birthday web app for your husband, featuring floating balloons, confetti, and a birthday journey experience.

## Features

- ✨ Animated floating balloons
- 🎊 Confetti effects
- ❤️ Heartbeat animation
- 💝 Glowing birthday message
- 🎈 Interactive "Start the Journey" button
- 📱 Responsive design for mobile and desktop

## Files

- `index.html` - Main birthday page with balloons and button
- `style.css` - Styling and animations
- `journey.html` - Placeholder for the birthday journey (ready for expansion)

## How to Deploy to GitHub Pages

### Step 1: Create a GitHub Repository

1. Go to [GitHub](https://github.com) and log in
2. Click the "+" icon in the top right and select "New repository"
3. Name your repository (e.g., "happy-birthday-love" or "birthday-surprise")
4. Make it **Public** (required for free GitHub Pages)
5. Click "Create repository"

### Step 2: Push Your Code to GitHub

Open your terminal in this project directory and run these commands:

```bash
# Initialize git repository
git init

# Add all files
git add .

# Commit the files
git commit -m "Initial birthday app commit"

# Add your GitHub repository as remote (replace USERNAME and REPO-NAME)
git remote add origin https://github.com/USERNAME/REPO-NAME.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click on "Settings" tab
3. Scroll down to "Pages" in the left sidebar
4. Under "Source", select "main" branch
5. Click "Save"
6. Wait a few minutes for deployment

### Step 4: Access Your Site

Your site will be available at:
```
https://USERNAME.github.io/REPO-NAME/
```
